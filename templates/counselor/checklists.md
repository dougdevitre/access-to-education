# College Planning Checklist (Junior/Senior)

**Student:** ___________________________ **Grade:** _____ **Counselor:** ___________________________
**Date:** _______________

## Junior Year

### Fall
- [ ] Review transcript — on track for graduation? credits, GPA, course rigor
- [ ] Complete career assessment on Missouri Connections (missouriconnections.org)
- [ ] Research colleges aligned to interests, academic profile, and budget
- [ ] Begin visiting campuses (virtual or in-person)
- [ ] Register for PSAT/NMSQT (October)
- [ ] Explore AP, IB, dual credit options for senior year

### Spring
- [ ] Take the ACT (free statewide administration for all juniors)
- [ ] Consider retaking ACT or taking SAT for higher scores
- [ ] Attend college fairs and financial aid information nights
- [ ] Begin scholarship search (fastweb.com, Missouri Connections, local)
- [ ] Meet with counselor to narrow college list (reach / match / safety)
- [ ] Plan summer activities (college visits, job, volunteer, enrichment)
- [ ] Check A+ eligibility status with A+ coordinator

## Senior Year

### August - October
- [ ] Finalize college list (3-8 schools recommended)
- [ ] Complete applications (Common App, individual apps)
- [ ] **FAFSA opens October 1** — complete ASAP (MO priority deadline: ~Feb 1)
- [ ] Request transcripts from counselor (allow 2 weeks)
- [ ] Ask 2-3 teachers for recommendation letters (give 3+ weeks notice)
- [ ] Write personal statement / application essays
- [ ] Retake ACT/SAT if needed (fall dates)
- [ ] Apply for early action / early decision if applicable (Nov 1-15 typical)
- [ ] Submit A+ eligibility paperwork

### November - January
- [ ] Complete remaining applications (regular decision deadlines: Jan 1 - Feb 1 typical)
- [ ] Submit FAFSA (if not already done)
- [ ] Apply for local scholarships (check with counselor for deadline list)
- [ ] Send ACT/SAT scores to colleges (use free score sends if possible)
- [ ] Check application status portals — ensure all materials received
- [ ] Complete any scholarship applications

### February - April
- [ ] Receive admission decisions
- [ ] Receive financial aid award letters — COMPARE:
  | College | COA | Grants/Scholarships | Loans | Net Cost |
  |---------|-----|-------------------|-------|----------|
  | | | | | |
  | | | | | |
- [ ] Visit finalist campuses
- [ ] Appeal financial aid if needed (write a professional appeal letter)
- [ ] Attend accepted students events

### May - June
- [ ] **Commit by May 1** — submit enrollment deposit to chosen college
- [ ] Decline other offers (so waitlisted students can have your spot)
- [ ] Complete housing application
- [ ] Register for orientation
- [ ] Send final transcript after graduation
- [ ] Thank your recommenders!
- [ ] File A+ final verification with coordinator

# Crisis Screening Quick Reference

**For school counselors conducting suicide risk screening or crisis triage.**

## When to Screen
- Student discloses suicidal thoughts, plans, or intent
- Staff, peer, or parent reports concerns about suicidality
- Student presents with self-harm injuries
- Student communicates hopelessness, "wanting to go away," or death-related content
- Social media or written content raises concern
- After a peer suicide or community trauma (monitor at-risk students)

## Screening Tools
| Tool | Use | Notes |
|------|-----|-------|
| **C-SSRS (Columbia Suicide Severity Rating Scale)** | Screening for suicidal ideation and behavior | Free; widely validated; recommended by DESE |
| **ASQ (Ask Suicide-Screening Questions)** | Brief 4-question screen for youth | Free; developed by NIMH; validated for ages 10-24 |
| **PHQ-A (Patient Health Questionnaire — Adolescent)** | Depression screening | May supplement suicide-specific screening |

## Response Protocol (After Screening)
| Risk Level | Indicators | Immediate Actions |
|-----------|-----------|------------------|
| **Imminent** | Active plan + means + intent; recent attempt | Do NOT leave alone → call 911 → contact parent → crisis services |
| **High** | Suicidal ideation with plan but no immediate intent; history of attempts | Do NOT leave alone → notify admin → contact parent → safety plan → community referral → same-day follow-up |
| **Moderate** | Suicidal ideation without plan; risk factors present | Notify admin → contact parent → safety plan → referral → next-day follow-up |
| **Low** | Fleeting thoughts; no plan, intent, or means; protective factors strong | Document → monitor → counseling check-ins → contact parent (use judgment on timing) |

## What NOT to Do
- Never promise to keep it a secret
- Never leave a high-risk student alone
- Never dismiss or minimize the concern
- Never use AI tools for clinical risk assessment
- Never delay action while waiting for "more information"

## Resources
- **988 Suicide & Crisis Lifeline:** call or text 988
- **Crisis Text Line:** text HOME to 741741
- **Missouri DMH crisis line:** varies by region
- **Trevor Project (LGBTQ+ youth):** 1-866-488-7386
