# Specialist Templates — IEP Meeting Prep & Progress Monitoring

## Template 1: IEP Meeting Preparation Checklist

**Student:** ___________________________ **Meeting Date:** _______________
**Meeting Type:** ☐ Initial ☐ Annual Review ☐ Triennial ☐ Amendment ☐ Transition ☐ MDR
**Case Manager:** ___________________________

### 2+ Weeks Before
- [ ] Send parent invitation (Prior Written Notice of meeting)
  - Includes: purpose, date, time, location, who will attend, parent rights
  - Sent via: ☐ Mail ☐ Email ☐ Hand-delivered ☐ Interpreter arranged (language: ___)
- [ ] Confirm parent can attend — offer at least 2 alternative dates/times if needed
- [ ] Invite all required IEP team members:
  - [ ] Parent/guardian
  - [ ] General education teacher
  - [ ] Special education teacher
  - [ ] LEA representative (can make resource commitments)
  - [ ] Person who can interpret evaluation results (if initial/triennial)
  - [ ] Student (required by age 16 for transition; invited earlier when appropriate)
  - [ ] Related service providers: _______________
  - [ ] Other agency representatives (transition): _______________
- [ ] Send teacher input forms to all general ed teachers
- [ ] Request excusal forms if any required member cannot attend (parent must agree in writing)

### 1 Week Before
- [ ] Collect and review all data:
  - [ ] Current grades and attendance
  - [ ] Progress on current IEP goals (most recent progress reports)
  - [ ] Teacher input forms returned
  - [ ] Behavioral data (if applicable)
  - [ ] Assessment results (if triennial/initial)
  - [ ] Transition assessment results (if age 14+)
  - [ ] Related services logs
  - [ ] State assessment results (MAP/EOC with accommodations)
- [ ] Draft proposed IEP updates:
  - [ ] Present levels of performance (PLAAFP) updated with current data
  - [ ] Proposed goals aligned to assessed needs
  - [ ] Proposed services, frequency, duration, location
  - [ ] Proposed accommodations and modifications
  - [ ] Transition plan updates (if applicable)
- [ ] Prepare parent-friendly summary of progress
- [ ] Print copies of: current IEP, proposed draft, procedural safeguards
- [ ] Reserve meeting room

### Day of Meeting
- [ ] Set up room — seating for all team members, copies at each seat
- [ ] Have blank forms ready (in case new goals or services are added)
- [ ] Start on time; introduce all team members
- [ ] Review purpose of the meeting
- [ ] Ensure parent is an active participant (not just a listener)
- [ ] Document all decisions — who proposed, who agreed, any disagreements

### Immediately After
- [ ] Provide parent a copy of the finalized IEP (or indicate when they'll receive it)
- [ ] Provide Procedural Safeguards if not given at meeting start
- [ ] Get parent consent for initial placement (initial IEPs only)
- [ ] Distribute IEP to all service providers within 5 school days:
  - [ ] General education teacher(s)
  - [ ] Special education teacher(s)
  - [ ] Related service providers
  - [ ] Paraprofessionals
  - [ ] Administrator
- [ ] Update SIS / IEP system with new dates, services, goals
- [ ] Calendar the next annual review date (within 365 days)
